# AssetVerse Client

## Project Name
**AssetVerse Client**

## Purpose
This is the frontend client for **AssetVerse**, built with React, TailwindCSS, and Vite. It provides a responsive user interface for managing assets, displaying charts, handling forms, and integrating with Firebase and Stripe for authentication and payments.

## Key Features
- Responsive UI built with **React** and **TailwindCSS**
- Client-side routing with **React Router**
- Data fetching and caching with **React Query**
- Form handling with **React Hook Form**
- Charts and visualizations with **Recharts**
- Animations with **Framer Motion** and **Lottie**
- PDF generation with **jsPDF** and **jsPDF-Autotable**
- Sweet alert notifications with **SweetAlert2**
- Carousel/slider functionality with **Swiper**
- Printing functionality with **react-to-print**
